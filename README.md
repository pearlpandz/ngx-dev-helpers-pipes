## NPM Package

[install npm package](https://www.npmjs.com/package/@ngx-dev-helpers/pipes)

```typescript
npm i @ngx-dev-helpers/pipes
```

## Import in module

```typescript
import { PipesModule } from '@ngx-dev-helpers/pipes';

@NgModule({
  imports: [ PipesModule ]
})
export class AppModule { }
```
